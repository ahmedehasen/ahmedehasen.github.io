3.Digit Recognition Based On FPGA Board

In this work, we achieve an efficient and accurate implementation of CNN on FPGA for a handwritten number recognition system. Initial training is done on computer itself and forward computation is implemented on FPGA. The proposed CNN was verified and compared with several CNN networks and, which gave the accuracy of about 98.2%. This network was implemented on Xilinx board XC7A35T-1CSG324C.

Publication These are my publications so far as a first author.

1. Towards Automatic Ethiopian Endemic Animals Detection on Android Using Deep Learning

This paper utilized You only look once (YOLO) v3 algorithm for animal detection. In This work, we provide large-scale animals dataset for the object detection task. The animal’s dataset consists six meaningful classes: chilada baboon, red fox, walia ibex, bale mountain vervet, black maned lion, and Somali wild ass. In addition, We have developed user friendly graphical user interface (GUI) based mobile android application. And, The results demonstrate that the proposed approach is highly efficient and effective which provides accuracy of 97.24% on our animal dataset.

2.Efficient Face Mask Detection Method Using YOLOX, An Approach to Reduce Coronavirus Spread

This paper proposes replacing of manual inspection of face mask detection with a deep learning method using YOLOX. In this work we have built a large scale datasets for Face mask detection by appling various Data augmentation techniques to improve the robustness of the model. Furthermore, we have added one more class that is an incorrect way of wearing masks so, covering only nose or mask is considered to be an incorrect way of wearing a mask. And, The results of the experiments show that the algorithm described in this work can efficiently detect face masks and enable more effective personnel monitoring than other YOLO series models. In addition, as compared to other models, our technique has significant benefits in terms of speed and accuracy in small and crowded areas.
